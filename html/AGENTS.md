# AI Agent Guide for Basics_Practice

This repository is a small static HTML practice project. It contains simple example pages exploring headings, paragraphs, links, and basic markup.

## What matters
- Project type: static HTML only
- No JavaScript, CSS frameworks, build tools, or packaging
- Files to focus on: `index.html`, `index2.html`, `H_links.html`, `lyrics.html`
- Inline comments in the HTML are intentional learning cues

## How to help
- Keep suggestions simple and beginner-friendly
- Fix HTML with plain semantic tags and valid structure
- Do not introduce advanced toolchains or unnecessary files
- Preserve learning comments unless the user explicitly asks to remove them

## When editing
- Use clear, minimal HTML improvements
- Prefer semantic elements like `<h1>`, `<p>`, `<a href="...">`, `<ul>`, `<li>`, etc.
- Avoid adding scripting, bundlers, or unrelated dependencies

## If asked to document
- Add only lightweight guidance relevant to HTML basics
- Link to existing repository files when helpful

## What not to do
- Do not assume this is a full web app
- Do not create complex frontend architecture
- Avoid adding tooling that does not match the current simple practice scope
